# 常见问题

> 这里整理 PaPiCode 使用过程中最常见的问题。

## 为什么我的客户端请求失败？

请检查 Base URL 是否为 `https://www.papicure.de/v1`，API Key 是否正确，账户是否有可用余额或订阅额度。

## 余额和订阅哪个先扣？

系统会根据账户计费偏好和可用额度选择扣费来源。通常有可用订阅时优先使用订阅额度。

## 支持哪些客户端？

理论上支持大多数 OpenAI Compatible 客户端，包括 Cursor、Chatbox、Cherry Studio，以及支持自定义端点的 CLI 工具。
