# AI 连接一切,Token 赋能未来

> PaPiCode 是面向开发者和团队的 AI API 中转与额度分发平台,支持 OpenAI Compatible 客户端、API Key 管理、Token 额度计费、套餐订阅与兑换码充值。

<div class="pp-hero-actions">

[开始接入](/guide/quickstart){.pp-btn-primary} [进入控制台](https://www.papicure.de){.pp-btn-secondary}

</div>

<CardGrid :cols="3">
  <FeatureCard title="统一 API 网关" eyebrow="网关">
    通过 OpenAI Compatible 接口接入多类模型,便于在 Cursor、Claude Code、Codex CLI 等工具中使用。
  </FeatureCard>
  <FeatureCard title="Token 额度计费" eyebrow="计费">
    以额度方式管理调用成本,支持余额、订阅套餐和兑换码等模式。
  </FeatureCard>
  <FeatureCard title="安全密钥管理" eyebrow="安全">
    每个用户可以创建多个 API Key,按用途隔离客户端和项目调用。
  </FeatureCard>
</CardGrid>

## 核心配置

| 字段 | 值 |
| --- | --- |
| **Base URL** | `https://www.papicure.de/v1` |
| **API Key** | 在控制台创建的密钥 |
| **协议** | OpenAI Compatible |

::: tip 使用提示
PaPiCode 面向海外用户和海外华人开发者。请根据您所在地区法律法规、上游服务条款和平台规则合规使用。
:::
