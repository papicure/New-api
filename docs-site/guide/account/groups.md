# 分组与线路说明

> PaPiCode 使用分组来区分不同的使用体验。用户无需理解内部计费细节,只需根据套餐和控制台提示使用即可。

<CardGrid :cols="3">
  <FeatureCard title="标准线路" eyebrow="日常调用">
    适合日常 API 调用,稳定可靠,适配大多数客户端和场景。
  </FeatureCard>
  <FeatureCard title="优选线路" eyebrow="高频场景">
    适合高频使用场景,兼顾稳定性与调用成本,首选给生产环境使用。
  </FeatureCard>
  <FeatureCard title="尊享线路" eyebrow="团队协作">
    适合重度开发者和团队协作,提供更高规格的使用体验和优先支持。
  </FeatureCard>
</CardGrid>
