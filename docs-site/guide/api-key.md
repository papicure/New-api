# 创建和管理 API Key

> API Key 是客户端调用 PaPiCode 的凭证。建议为不同客户端或项目分别创建独立密钥。

## 创建步骤

**进入控制台**

登录后打开 API Key / 令牌管理页面。

**新建密钥**

填写名称，例如 Cursor、Codex CLI、Chatbox 或项目名。

**保存密钥**

密钥只展示一次，请保存到安全位置。

## 建议实践

### 按项目隔离

为不同项目或客户端创建不同 Key，便于排查和停用。

### 定期轮换

如果密钥泄露或项目结束，请及时禁用或删除。

### 避免前端暴露

不要把密钥写入公开网页、前端仓库或可被用户查看的配置。
