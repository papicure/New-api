# 在 Chatbox 中使用 PaPiCode

> Chatbox 支持自定义 OpenAI API Host,适合日常聊天、模型测试和轻量办公。

<div class="steps">

1. **打开设置** — 进入 Model Provider 或 API 设置。
2. **选择 OpenAI API** — 将 API Host 改为 PaPiCode 地址。
3. **填写地址** — `https://www.papicure.de/v1`
4. **填写密钥** — API Key 填写 PaPiCode 控制台创建的密钥。

</div>
