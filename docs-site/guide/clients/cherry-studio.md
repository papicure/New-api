# 在 Cherry Studio 中使用 PaPiCode

> Cherry Studio 支持添加自定义 OpenAI Compatible 服务商。

<div class="steps">

1. **添加服务商** — 选择 OpenAI Compatible 或自定义服务商。
2. **填写接口地址** — `https://www.papicure.de/v1`
3. **填写密钥** — 使用 PaPiCode 控制台创建的 API Key。
4. **同步模型** — 如客户端支持模型列表同步,可尝试刷新模型列表。

</div>
