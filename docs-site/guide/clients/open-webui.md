# 在 Open WebUI 中使用 PaPiCode

> Open WebUI 可以通过 OpenAI Compatible 方式接入 PaPiCode，适合团队内部搭建统一聊天界面。

## 环境变量方式

```
OPENAI_API_BASE_URL=https://www.papicure.de/v1
OPENAI_API_KEY=YOUR_API_KEY
```

## 管理后台方式

在 Open WebUI 管理后台添加 OpenAI Compatible 连接，填写 Base URL 和 API Key。
