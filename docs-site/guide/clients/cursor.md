# 在 Cursor 中使用 PaPiCode

> Cursor 支持配置兼容 OpenAI 的 API 地址。将 PaPiCode 作为自定义供应商即可使用。

**打开 Cursor 设置**

进入 Models 或 API Keys 配置区域。

**选择 OpenAI Compatible**

添加自定义服务商或覆盖 OpenAI Base URL。

**填写 Base URL**

`https://www.papicure.de/v1`

**填写 API Key**

复制 PaPiCode 控制台创建的 API Key。

**选择模型并测试**

保存后发起一次简单对话，确认调用成功。

**建议**

为 Cursor 单独创建一个 API Key，便于后续查看用量或单独停用。
