# Frequently asked questions

> Common questions about using PaPiCode.

## Why does my client request fail?

Check the Base URL, API key, available credits, and model name.

## Which funding source is used first?

When an active subscription has credits, it is usually used before wallet balance.

## Which clients are supported?

Most OpenAI-compatible clients, including Cursor, Chatbox, Cherry Studio, and custom SDK integrations.
