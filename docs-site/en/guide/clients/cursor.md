# Use PaPiCode in Cursor

> Configure PaPiCode as an OpenAI-compatible custom provider in Cursor.

**Open Cursor settings**

Go to Models or API Keys settings.

**Select OpenAI Compatible**

Add a custom provider or override the OpenAI Base URL.

**Set Base URL**

`https://www.papicure.de/v1`

**Set API Key**

Paste the key created in the PaPiCode console.

**Test a model**

Save and send a simple prompt.
