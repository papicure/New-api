# Use PaPiCode in Cursor

> Configure PaPiCode as an OpenAI-compatible custom provider in Cursor.

<div class="steps">

1. **Open Cursor settings** — Go to Models or API Keys settings.
2. **Select OpenAI Compatible** — Add a custom provider or override the OpenAI Base URL.
3. **Set Base URL** — `https://www.papicure.de/v1`
4. **Set API Key** — Paste the key created in the PaPiCode console.
5. **Test a model** — Save and send a simple prompt to verify the call.

</div>

::: tip Recommendation
Create a dedicated API key for Cursor so usage can be tracked and revoked independently.
:::
