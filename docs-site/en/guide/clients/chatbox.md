# Use PaPiCode in Chatbox

> Chatbox can connect to custom OpenAI API hosts.

**Open settings**

进入 Model Provider 或 API 设置。

**Choose OpenAI API**

将 API Host 改为 PaPiCode 地址。

**Set endpoint**

`https://www.papicure.de/v1`

**Set key**

API Key 填写 PaPiCode 控制台创建的密钥。
