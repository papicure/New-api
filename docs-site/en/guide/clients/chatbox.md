# Use PaPiCode in Chatbox

> Chatbox can connect to custom OpenAI API hosts, which makes it a good fit for casual chat, model evaluation, and light office workflows.

<div class="steps">

1. **Open settings** — Go to Model Provider or API settings.
2. **Choose OpenAI API** — Change the API Host to point at PaPiCode.
3. **Set endpoint** — `https://www.papicure.de/v1`
4. **Set key** — Paste the API key created in the PaPiCode console.

</div>
