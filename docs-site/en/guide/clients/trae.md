# Use PaPiCode in Trae

> Add PaPiCode as a custom OpenAI-compatible model provider in Trae.

**Open AI settings**

Go to AI, Models, Model Provider, or custom model settings.

**Add custom model**

Select OpenAI Compatible or Custom OpenAI Endpoint.

**Set endpoint**

Use `https://www.papicure.de/v1`.

**Set key**

Use your PaPiCode API key.

**Set model name**

Copy the exact model name from the PaPiCode console.

**Tip**

Name the provider clearly, for example PaPiCode GPT or PaPiCode Claude, to avoid selecting the wrong provider.
