# Use PaPiCode in CodeBuddy

> If your CodeBuddy version supports custom models, OpenAI Compatible providers, or custom API endpoints, configure PaPiCode as a provider.

**Open model settings**

Find Model, Provider, API Key, Custom Model, or OpenAI Compatible settings.

**Add a custom provider**

Select OpenAI Compatible. If unavailable, choose OpenAI and override the Base URL.

**Set Base URL**

Use `https://www.papicure.de/v1`. Do not append `/chat/completions`.

**Set API Key**

Paste the API key created in the PaPiCode console.

**Test a model**

Use a model name listed in your console and send a short prompt.

Provider Type`OpenAI Compatible`

Base URL`https://www.papicure.de/v1`

API Key`YOUR_API_KEY`

Model`Available model in console`
