# Redeem codes

> PaPiCode currently recommends redeem codes for wallet top-ups.

**Get a redeem code**

Use the purchase entry shown in the wallet page.

**Open wallet**

Sign in and go to Wallet / Top-up.

**Redeem**

Submit the code and credits will be added to your balance.
