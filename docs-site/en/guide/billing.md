# How wallet and subscription credits are consumed

> PaPiCode uses token credits for billing. Wallet balance and subscription credits are separate funding sources.

Type

Description

Use case

Wallet

Credits obtained through redeem codes.

Occasional usage and top-ups.

Subscription

A dedicated credit pool within a validity period.

Recurring usage and teams.

Groups

Different usage tiers and route experiences.

Higher-frequency or team workflows.
