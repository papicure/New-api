# Create and manage API keys

> API keys authenticate client requests. Create separate keys for different tools and projects.

### Separate by project

Create different keys for Cursor, Codex CLI, automation scripts, and backend services.

### Rotate regularly

Disable or delete keys when a project ends or a key might be exposed.

### Keep keys private

Never commit keys to public repositories or frontend code.
